# Steem Apps Explorer

## Project setup
```
npm install
```

### Development build
```
npm run serve
```

### Production build
```
npm run build
```
### Config
Please add the following api endpoints to your .env file (root folder):
```
VUE_APP_HIVEMIND_API = 'https://tower.emrebeyler.me/api/v1'
VUE_APP_STEEMIT_MAINNET_PROD = 'https://api.steemit.com'
```
